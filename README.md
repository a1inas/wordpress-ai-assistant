# AI Chat Assistant для WordPress

**Версия:** 1.0.0  
**PHP:** 7.4+  
**WordPress:** 5.0+

AI-ассистент для WordPress с интеграцией Llama 3.1 через Groq API.

Плагин добавляет на сайт умного AI-консультанта, который отвечает на вопросы посетителей, помогает с выбором и собирает заявки для менеджеров.

---

## Возможности

### AI-ассистент
- Интеграция с **Llama 3.1** через Groq API
- Настройка модели, температуры и максимального количества токенов
- Учёт контекста текущей страницы для релевантных ответов
- История диалогов (хранится в БД)

### Гибкая настройка
- Админ-панель с полным управлением
- Настраиваемый **системный промпт** (стиль общения)
- Подробные **правила ассистента** (язык, тон, запреты)
- Загрузка **контекста сайта** (факты о компании)
- Выбор основного цвета под дизайн сайта

### Сбор заявок
- Кнопка «Связаться с менеджером»
- Форма с валидацией полей (имя, email, телефон, время)
- Автоматическая отправка заявки на email менеджера
- Хранение лидов в базе данных

### База данных
- Таблица для истории сообщений
- Таблица для заявок
- Автоматическая очистка старых записей (CRON)

### Дизайн
- Адаптивный интерфейс (десктоп / мобильные)
- Кастомизация через CSS-переменные
- Анимации и индикатор печати
- Поддержка цветовой схемы сайта

## Установка

### Через админку WordPress
1. Скачайте архив с плагином
2. Перейдите в **Плагины → Добавить новый → Загрузить плагин**
3. Выберите архив и нажмите «Установить»
4. Активируйте плагин

### Вручную (через FTP)
1. Скопируйте папку `ai-chat-assistant` в `/wp-content/plugins/`
2. Зайдите в админку WordPress
3. Перейдите в **Плагины** и активируйте **AI Chat Assistant**

---

## Настройка

После активации перейдите в **AI Chat Assistant → Настройки**:

1. **Groq API Key** — вставьте свой ключ (получить можно на [console.groq.com](https://console.groq.com))
2. **Модель** — выберите модель (по умолчанию `llama3-8b-8192`)
3. **Температура** — настройте креативность ответов (0.1 — строго, 1.5 — креативно)
4. **Системный промпт** — задайте стиль общения
5. **Правила ассистента** — подробные инструкции (язык, тон, запреты)
6. **Контекст сайта** — добавьте информацию о компании
7. **Email для уведомлений** — куда приходят заявки

## Использование

### На любой странице
Чат автоматически появится в правом нижнем углу сайта.

### Кнопка «Связаться с менеджером»
При нажатии открывается форма сбора заявки. После отправки:
- Данные сохраняются в БД
- Менеджер получает email-уведомление
- Пользователь видит подтверждение

### Обработка диалогов
- История хранится 30 дней (настраивается)
- Автоматическая очистка через CRON
- В админке можно просмотреть последние 50 сообщений

## Получение API-ключа

1. Зарегистрируйтесь на [console.groq.com](https://console.groq.com)
2. Перейдите в раздел **API Keys**
3. Нажмите **Create API Key**
4. Скопируйте ключ (начинается с `gsk_...`)
5. Вставьте в настройки плагина

## База данных

### Таблица `wp_ai_chat_messages`
- `id` — уникальный ID сообщения
- `session_id` — ID сессии пользователя
- `role` — `user` или `assistant`
- `message` — текст сообщения
- `created_at` — дата и время

### Таблица `wp_ai_chat_leads`
- `id` — уникальный ID заявки
- `name` — имя пользователя
- `email` — email
- `phone` — телефон
- `contact_time` — удобное время для связи
- `message` — сообщение
- `status` — статус заявки (`new`)
- `created_at` — дата и время

## Правила ассистента (пример)

В плагин уже загружены подробные правила:
- Отвечать только на русском
- Не использовать эмодзи и Markdown
- Не выдумывать факты
- При вопросах о цене/сроках предлагать форму
- Не давать контакты без прямого запроса
